# Initiative: Tracker Binding Context Discovery

This directory packages the follow-on implementation prompts for the three
repositories affected by ADR
`2026-04-04-1-tracker-binding-context-is-discovered-not-user-supplied.md`.

These files are not ADRs. They are copy/paste prompt sources for
`/spec-kitty.specify` in each repository.

Files:

1. `spec-kitty-team-prompt.md`
2. `spec-kitty-saas-team-prompt.md`
3. `spec-kitty-tracker-team-prompt.md`

Use each prompt in the corresponding repository so the resulting feature spec
stays repo-local while sharing one architectural decision.
